# twind

---

## READ THIS FIRST!

**Twind v1 is still in beta. Expect bugs!**

Twind v1 is a complete rewrite aiming to be compatible with Tailwind v3 classes

---

## Installation

Install from npm:

```sh
# Using npm
npm install twind@next

# Using Yarn
yarn add twind@next
```

## Usage

```js
import 'twind'
```

Or using a CDNs:

```html
<script src="https://cdn.jsdelivr.net/npm/twind@next"></script>
```
