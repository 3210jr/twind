# @twind/runtime

---

## READ THIS FIRST!

**Twind v1 is still in beta. Expect bugs!**

---

Manages a single Twind instance which observes the DOM.

## Installation

Install from npm:

```sh
# Using npm
npm install @twind/runtime@next

# Using Yarn
yarn add @twind/runtime@next
```

## Usage

```js
import { init } from '@twind/runtime'

init({
  /* twind config */
})
```
