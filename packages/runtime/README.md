## Installation

Install from npm:

```sh
# Using npm
npm install @twind/runtime@next

# Using Yarn
yarn add @twind/runtime@next
```

## Usage

```js
import { setup } from '@twind/runtime'

// can be called multiple times
setup({
  /* twind config */
})
// -> returns a Twind instance eg `tw`
```
